```shell
npm init gatsby
```

- Start by going to the directory with
```shell
cd my-gatsby-site
```

- Start the local development server 
```shell
npm run develop
```

### Run your site locally

생성이 완료되면 해당 폴더로 이동하고 실행을 시켜본다. 만약 gatsby-cli가 global로 설정되었다면 gatsby develop을 하고 아니면 npm run develop을 한다.
