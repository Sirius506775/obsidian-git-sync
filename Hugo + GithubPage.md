##### Hugo를 이용한 GithubPage 만들기
1. Hugo 설치
	1. OS에 맞는 최신 버젼 다운로드
	2. C:\Hugo\bin 디텍토리를 생성해서 다운받은 압축파울 해제
	3. Gitbub 저장소를 2개 만든다. 
		1. Hugo의 컨텐츠와 소스파일을 포함할 repo
		2. 렌더링된 버젼의 Hugo 웹 사이트를 포함하는 github.io

4. prompt 창에서 $ hugo new site blog를 입력해서 디렉토리 구조를 만든다.
