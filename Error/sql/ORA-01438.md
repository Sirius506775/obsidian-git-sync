ORA-01438:이 열에 대해 지정된 전체 자릿수보다 큰 값이 허용됩니다.
ORA-01438:value larger than specified precision allowed for this column

입력하려는 데이터가 해당 컬럼의 크기와 맞지 않기 때문에 생기는 에러

=> 데이터를 컬럼에 맞게 수정하거나, 테이블의 컬럼 크기를 변경해야한다.
